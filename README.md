# 📌 파일명 일괄 변경 프로그램 (v1.0)

**파일명 일괄 변경 프로그램**은 과거 윈도우 필수 유틸리티였던 'DarkNamer'의 직관적인 사용성을 현대적인 웹 기술로 재현한 고성능 파일 관리 도구입니다. 복잡한 설치 없이 브라우저에서 즉시 수백 개의 파일 이름을 규칙에 따라 일괄 변경하고, 안전하게 ZIP으로 내보낼 수 있습니다.

---

## ✨ 핵심 기능

### 1. 강력한 이름 변경 규칙 (Name Rules)
- **문자열 바꾸기**: 특정 단어를 찾아 다른 단어로 한 번에 교체합니다.
- **앞이름/뒷이름 붙이기**: 파일명의 시작이나 끝에 원하는 텍스트를 추가합니다.
- **이름 지우기**: 확장자를 제외한 본래의 파일명을 완전히 삭제합니다.
- **괄호 제거**: `[ ]`, `( )`, `{ }` 등 모든 종류의 괄호와 그 내부 텍스트를 즉시 제거합니다.
- **숫자 제어**: 숫자만 남기거나, 숫자만 골라 삭제할 수 있습니다.
- **자릿수 맞추기 (Padding)**: 파일명 길이를 통일하기 위해 앞에 '0'을 채워 넣습니다.
- **번호 매기기 (Numbering)**: 시작 번호와 자릿수를 지정하여 순차적인 번호를 부여합니다.

### 2. 정밀한 확장자 관리 (Extension Control)
- 확장자만 일괄 삭제하거나, 새로 추가하거나, 다른 형식으로 변경할 수 있습니다.

### 3. 스마트 사용자 경험 (v1.0 신기능)
- **실시간 중복 감지**: 동일한 파일명이 발생하면 **로즈(Rose) 색상 하이라이트**와 경고 아이콘을 통해 즉시 알립니다.
- **안전한 내보내기**: 중복된 이름이 존재할 경우 다운로드를 제한하여 파일 덮어쓰기 사고를 원천 방지합니다.
- **엔터(Enter) 단축키**: 설정 모달에서 입력 후 엔터키를 누르면 즉시 '적용' 버튼을 누른 것과 동일하게 작동하여 작업 속도를 높여줍니다.
- **타임스탬프 다운로드**: ZIP 파일 추출 시 `파일명_일괄_변경_YYYYMMDD_HHMMSS.zip` 형식으로 현재 시간이 자동 기록됩니다.
- **드래그 앤 드롭 (Drag & Drop)**: 파일을 직접 끌어다 놓거나, 목록 내에서 파일 순서를 드래그하여 자유롭게 변경할 수 있습니다.

## 🔒 보안 및 개인정보 보호 (Privacy First)

본 도구는 사용자의 데이터를 서버로 전송하지 않는 **완전 로컬 처리 방식**입니다.
- **100% Client-Side**: 모든 연산은 서버가 아닌 사용자의 브라우저 메모리 내에서 수행됩니다.
- **오프라인 작동 가능**: 파일은 외부로 유출되지 않으며, 인터넷 연결 없이도 안전하게 작동합니다.
- **중요 문서 보호**: 민감한 문서나 개인 사진의 파일명을 외부 유출 걱정 없이 안심하고 처리하세요.

## 🚀 사용 방법

1. **파일 업로드**: `파일 추가` 버튼을 누르거나 파일을 화면 중앙으로 끌어다 놓습니다.
2. **순서 변경**: 목록 왼쪽의 번호 영역을 드래그하여 파일의 처리 순서를 변경할 수 있습니다.
3. **규칙 적용**: 왼쪽 사이드바에서 원하는 기능을 선택하고 필요한 설정값을 입력합니다. (입력 후 **Enter** 키로 빠른 적용 가능)
4. **미리보기 및 검토**: '바뀔 이름' 컬럼을 실시간으로 확인합니다. 중복(빨간색)이 발생하면 번호 붙이기 등으로 수정합니다.
5. **작업 취소**: 실수했다면 하단의 `작업 취소` 버튼을 눌러 이전 상태로 되돌립니다.
6. **내보내기**: `다운로드(ZIP)` 버튼을 클릭하여 이름이 변경된 새 파일들을 압축 파일로 저장합니다.
