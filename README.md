
# 📌 파일명 일괄 변경툴: 프로페셔널 도구

**파일명 일괄 변경툴**은 과거 윈도우용 필수 유틸리티였던 'DarkNamer'의 직관적인 사용성을 웹 환경에서 재현한 고성능 파일 이름 변경 도구입니다. 복잡한 설치 과정 없이 브라우저에서 즉시 수백 개의 파일 이름을 규칙에 따라 일괄 변경하고 ZIP으로 다운로드할 수 있습니다.

---

## ✨ 핵심 기능

### 1. 다양한 이름 변경 규칙
- **문자열 바꾸기**: 특정 단어를 찾아 다른 단어로 일괄 교체합니다.
- **앞/뒷이름 붙이기**: 파일명 시작 부분이나 끝 부분에 특정 텍스트를 추가합니다.
- **이름 지우기**: 파일명을 완전히 비우고 확장자만 남깁니다 (번호 붙이기와 조합 시 유용).
- **괄호 제거**: `[ ]`, `( )`, `{ }` 등 모든 종류의 괄호와 그 안의 내용을 한 번에 삭제합니다.
- **숫자만 남기기**: 파일명에서 숫자를 제외한 모든 문자를 제거합니다.
- **자릿수 맞추기 (Padding)**: 파일명 길이를 맞추기 위해 앞에 '0'을 채워 넣습니다.
- **번호 매기기**: 시작 번호와 자릿수를 지정하여 파일에 순차적인 번호를 부여합니다.

### 2. 강력한 확장자 관리
- 확장자 일괄 삭제, 추가, 변경 기능을 지원합니다.

### 3. 직관적인 사용자 경험 (UX)
- **드래그 앤 드롭 (Drag & Drop)**: 파일을 브라우저로 직접 끌어다 놓아 즉시 추가할 수 있습니다.
- **실시간 미리보기**: 기능을 적용하는 즉시 '바뀔 이름' 컬럼을 통해 결과를 확인합니다.
- **작업 취소 (Undo)**: 실수로 잘못 적용했을 경우 이전 상태로 되돌릴 수 있습니다.
- **반응형 테이블**: 파일 목록을 깔끔하게 정리하여 보여주며, 변경된 부분은 파란색으로 강조됩니다.

---

## 🔒 보안 및 개인정보 보호

본 도구는 사용자의 데이터 보안을 최우선으로 생각합니다.
- **100% 로컬 처리**: 모든 파일 연산은 서버가 아닌 **사용자의 웹 브라우저 내(Local)**에서 수행됩니다.
- **서버 전송 없음**: 파일은 외부 서버로 절대 업로드되지 않습니다. 인터넷 연결이 끊긴 상태에서도 작동합니다.
- **개인정보 안전**: 소중한 문서나 사진의 파일명이 외부로 유출될 걱정 없이 안심하고 사용하세요.

---

## 🚀 사용 방법

1. **파일 추가**: 상단의 `파일 추가` 버튼을 클릭하거나, 변경하고 싶은 파일들을 브라우저 화면으로 직접 끌어다 놓습니다.
2. **규칙 적용**: 왼쪽 사이드바에서 원하는 변경 기능을 선택합니다.
3. **설정 입력**: 팝업창(모달)이 뜨면 필요한 옵션(찾을 단어, 번호 시작 등)을 입력하고 `적용`을 누릅니다.
4. **미리보기**: 테이블의 '바뀔 이름'을 확인하며 원하는 결과가 나올 때까지 다른 규칙을 중첩해서 적용합니다.
5. **적용 및 저장**: 만족스럽다면 우측 상단의 `일괄 변경 적용 (ZIP 다운로드)` 버튼을 클릭합니다. 모든 파일이 변경된 이름으로 압축되어 다운로드됩니다.
